# Task 1 – Step 1: Ethical Hacking Rules & Scope

## What is Ethical Hacking?
Ethical hacking is the authorized practice of testing systems, networks, and applications to identify security vulnerabilities before malicious attackers can exploit them.

## Ethical Hacking vs Illegal Hacking

### Ethical Hacking
- Performed with permission.
- Conducted within a defined scope.
- Helps improve security.
- Findings are responsibly disclosed.

### Illegal Hacking
- Performed without authorization.
- Violates laws and regulations.
- Causes harm or unauthorized access.
- Data may be stolen or modified.

## Permission, Scope, and Responsible Disclosure

### Permission
Always obtain written authorization before conducting security testing.

### Scope
Only test systems that are explicitly included in the engagement.

### Responsible Disclosure
Report vulnerabilities privately to the organization and allow time for remediation before public disclosure.
