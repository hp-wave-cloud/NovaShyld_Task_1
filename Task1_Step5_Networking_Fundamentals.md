# Task 1 – Step 5: Networking Fundamentals

## IP Address
An IP address uniquely identifies a device on a network.

## Ports
Ports allow multiple services to run on a single device.
Examples:
- HTTP: 80
- HTTPS: 443
- SSH: 22
- DNS: 53

## TCP vs UDP

TCP:
- Connection-oriented
- Reliable
- Error checking
- Used for HTTP, HTTPS, SSH

UDP:
- Connectionless
- Faster
- No delivery guarantee
- Used for DNS, Streaming

## DNS Basics
DNS translates domain names into IP addresses.

Example:
google.com → 142.xxx.xxx.xxx

## HTTP vs HTTPS

HTTP:
- Port 80
- Unencrypted

HTTPS:
- Port 443
- Encrypted using SSL/TLS
